MacroBox v1.0

All files available on GitHub.

Overview

The MacroBox v1 is the first design in the MacroBox series.
It is the least budget efficient and least effective design as it is, after all the first design.

It contains a custom-designed PCB holding ten Cherry MX Blue Switches along with an ATmega32u4 as the brains of the system. Eight of the switches are programmed to simulate keystrokes and act as macros for any use. The top two switches change the board’s current “Mode” and macro “Page”, allowing the board to simulate more than eight macros. The current “Mode” is displayed by the first RGB LED, showing a custom colour for the mode, well the second LED displays the “Page”.


Printed Circuit Board

The PCB for this project is 1.6mm thick and consists of two layers.
The files for the PCB are available only on EasyEDA.


Code

The plan was to flash the chip with the Arduino Micro Firmware and program it in Arduino, with help from the Keyboard Library.

Unfortunately, I never got to that as I began working on a cheaper version, the MacroBox v2.
